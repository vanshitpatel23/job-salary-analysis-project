# job Salary Analysis Project

## 📌 Overview

This project focuses on analyzing job salary data using Python. It helps in identifying high-paying roles, salary trends, and insights using data visualization techniques.

---

## 🛠️ Tech Stack

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn

* Data visualization using charts
